# PQL Queries – WoodCorp O2C

## On-Time Delivery %
```sql
AVG(CASE WHEN DELIVERED_DATE <= PROMISED_DATE THEN 1 ELSE 0 END)
```

## Late Orders
```sql
SUM(CASE WHEN DELIVERED_DATE > PROMISED_DATE THEN 1 ELSE 0 END)
```

## Late Quantity
```sql
SUM(CASE WHEN DELIVERED_DATE > PROMISED_DATE THEN DELIVERED_QUANTITY END)
```

## Volume Conformance
```sql
AVG(CASE WHEN DELIVERED_QUANTITY BETWEEN ORDERED_QUANTITY + MIN_ORDER_TOLERANCE AND ORDERED_QUANTITY + MAX_ORDER_TOLERANCE THEN 1 ELSE 0 END)
```

## Avg Delay
```sql
AVG(DAYS_BETWEEN(PROMISED_DATE, DELIVERED_DATE))
```
